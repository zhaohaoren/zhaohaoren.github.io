变量定义

基本定义：使用var修饰，类型放在名称后

var a int = 1

如果知道初始化的值，go可以从初始化值类型推断

var a = 1 // 不指定类型，go推断类型

变量的初始化还可以进一步简写，使用 xxx := 等价于 var xxx type = （推荐，能少些东西就少些）

name := "justin"

不过该方式只能再func内部，不能func外使用

package main
nameOut := "justin"  // error -- 包级别变量不可以
func main() {
    nameIn := "justin"  // ok  -- 方法级别可以
}

可以多个变量定义在一行，其中类型也可以支持推断

var a, b, c = 1, 2, b
j, h, k := 1, 2, b

这种定义，少量变量还行，如果变量太多，或者需要注释看的清洗就不好了，Go又提供一种简写进行集中变量定义

var (
    n1 = 100
    n2 = 200
    n3 = "hello"
)
